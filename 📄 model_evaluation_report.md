# House Price Prediction – Model Evaluation Report

## Dataset
House Prices Dataset (Provided)

## Models Implemented
- Linear Regression
- Polynomial Regression
- Decision Tree Regressor
- Random Forest Regressor

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

## Results
- Linear Regression R² ≈ 0.78
- Random Forest achieved the best performance

## Key Insights
- Area is the strongest predictor of house price
- Location has a significant impact on pricing
- Tree-based models handle non-linear relationships better

## Conclusion
Random Forest is the most suitable model for accurate house price prediction.
